# 🌍 Street View Avatar - מדריך שימוש

## מה זה?

אפליקציה שמאפשרת לך לשלב את הדמות שלך בתוך תמונות Street View אמיתיות מכל מקום בעולם!

## 🚀 איך להתחיל?

### שלב 1: קבלת Google Maps API Key

1. היכנס ל-[Google Cloud Console](https://console.cloud.google.com/)
2. צור פרויקט חדש או בחר פרויקט קיים
3. עבור ל-**APIs & Services** > **Credentials**
4. לחץ על **Create Credentials** > **API Key**
5. העתק את ה-API Key
6. **חשוב:** הפעל את ה-APIs הבאים:
   - Maps JavaScript API
   - Street View Static API
   - Geocoding API

### שלב 2: פתיחת האפליקציה

1. פתח את הקובץ `streetview-avatar.html` בדפדפן
2. הדבק את ה-API Key בשדה המתאים בצד ימין

### שלב 3: שימוש באפליקציה

#### 🔍 חיפוש מיקום
- הכנס כתובת או שם מקום (לדוגמה: "Eiffel Tower, Paris")
- לחץ על "חפש מיקום"
- ה-Street View יעבור למיקום הנבחר

#### 🎮 ניווט במצלמה
השתמש בכפתורי הבקרה:
- **↑** - התקדם קדימה
- **↓** - זוז אחורה
- **← →** - זוז שמאלה/ימינה
- **↶ ↷** - סובב את המצלמה שמאלה/ימינה

אפשר גם לגרור במצלמה באמצעות העכבר!

#### 📸 העלאת דמות
1. לחץ על "בחר תמונת דמות"
2. בחר תמונה של עצמך (רצוי עם רקע שקוף PNG)
3. התמונה תופיע בתצוגה המקדימה
4. התאם את הגודל והשקיפות באמצעות המחוונים

#### ✨ יצירת תמונה משולבת
1. וודא שיש לך דמות טעונה
2. התאם את זווית המצלמה למיקום המדויק
3. לחץ על "צור תמונה משולבת"
4. התמונה המשולבת תופיע למטה
5. לחץ ימני על התמונה ובחר "שמור תמונה בשם" להורדה

#### 💾 שמירת מיקומים
- לחץ על "שמור מיקום זה" כדי לשמור את המיקום הנוכחי
- המיקומים השמורים יופיעו ברשימה בצד
- לחץ על מיקום שמור כדי לחזור אליו

## 🎨 טיפים לשימוש

### איכות תמונה
- השתמש בתמונות באיכות גבוהה של הדמות
- תמונות עם רקע שקוף (PNG) יראו הכי טוב
- הקפד שהדמות תהיה במרכז התמונה

### בחירת זווית
- נסה זוויות שונות של המצלמה
- מצא רחוב ריק יחסית לתוצאה ריאליסטית יותר
- שחק עם הגובה (pitch) של המצלמה

### שילוב ריאליסטי
- התאם את גודל הדמות לפרופורציה של הרחוב
- השתמש בשקיפות נמוכה יותר לאפקט עדין
- מצא תאורה דומה בתמונת הדמות וברחוב

## 🌟 פיצ'רים מתקדמים

### שילוב עם AI (אופציונלי)
בגרסאות עתידיות ניתן לשלב:
- **Stable Diffusion Inpainting** - לשילוב מציאותי יותר
- **Background Removal AI** - להסרת רקע אוטומטית
- **Style Transfer** - להתאמת סגנון התאורה

### API Integration
אפשר להוסיף אינטגרציה עם:

- Stability AI API

## ⚠️ הערות חשובות

1. **API Key** - שמור על ה-API Key שלך פרטי, אל תשתף אותו
2. **מכסות** - Google Maps API יש מכסת שימוש חינמית, אבל מוגבלת
3. **רשת** - נדרשת חיבור לאינטרנט יציב
4. **זכויות** - וודא שיש לך זכויות להשתמש בתמונות שאתה מעלה

## 🐛 פתרון בעיות

### Street View לא נטען
- וודא שה-API Key תקין
- בדוק שהפעלת את כל ה-APIs הנדרשים
- נסה לרענן את הדף

### המיקום לא נמצא
- נסה לכתוב את הכתובת באנגלית
- הוסף פרטים נוספים (עיר, מדינה)
- נסה קואורדינטות (Lat, Lng)

### התמונה לא משתלבת
- וודא שהעלית תמונת דמות
- בדוק שה-API Key תקין
- נסה להקטין את גודל התמונה
